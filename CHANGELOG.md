# Changelog

All notable changes to PERSCOM Advance will be documented here.

## [1.0.0] — 2026-03-15

### Initial Release

- Full personnel management system with ranks, roles, and service records
- Operations center with attendance and AAR tracking
- Evaluation system with scoring and reviewer assignments
- Discord bot integration with automatic role sync
- ORBAT (Order of Battle) management
- Document repository
- Gear loadout system
- Attendance tracking
- Dashboard with analytics, countdown timer, and personnel growth chart
- Spotlight carousel for featured personnel
- Customizable unit branding via `unit.config.json`
- License validation system
- Auto-update notifications
